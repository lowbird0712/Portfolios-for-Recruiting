"열받는 이세계(2020년)" 프로젝트의 전에 진행한 개인 프로젝트에 타일맵 에디터의 매뉴얼이 적힌 문서입니다.
이 프로젝트는 타일맵 위에서 캐릭터가 이동하며 타일맵 상의 오브젝트와 상호작용하면 이야기가 진행되는 스토리 시스템을 가지고 있습니다.
"イライラ異世界(2020年)"プロトタイプの前に進行した個人プロトタイプに使用されたタイルマップのエディターのマニュアルが書かれた文章です。
このプロジェクトはタイルマップの上でキャラクターが移動しながらタイルマップ上のオブジェクトと相互作用をすると物語が進行されるストーリーシステムを持っています

タイルマップシステム(타일맵 시스템)には

タイルマップシステム
構成要素				: タイル、オブジェクト、オプション、イベントトリガー、イベントトリガーノード
タイル					: 一番的なタイルマップのタイルそのものだ。
オブジェクト			: 配置できる構造物でタイル複数分のサイズを持つ場合もある。
オプション				: O(移動可)、X(移動不可)、三角(移動可、だがオブジェクトがプレーヤーの前に出力される)
イベントトリガー		: イベントトリガーが設定されたタイルの上にプレイヤーがいる時特定の条件を満足するとイベントが発生する。このイベントによってタイルマップが変化する場合もある。
イベントトリガーノード	: イベントトリガーが設定されたタイルマップ一つ一つを示して複数のノードがリストを構成している。タイルマップが変化するイベントが発生する時次にのタイルマップ
							に今のタイルマップを入りかえる。

と書いております
/////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////

타일맵 에디터 사용 매뉴얼

타일맵 시스템
 구성 요소			: 타일, 오브젝트, 옵션, 이벤트 트리거, 이벤트 트리거 노드
 타일				: 일반적인 타일맵의 타일 그 자체이다.
 오브젝트			: 배치 가능한 구조물로 타일 여러개의 크기를 가질 수도 있다.
 옵션				: O(이동 가능), X(이동 불가), 세모(이동 가능하나 오브젝트가 플레이어의 앞에 출력)
 이벤트 트리거		: 이벤트 트리거가 설정된 타일 위에서 플레이어가 있을 때 특정 조건을 만족하면 이벤트가 발생한다. 이 이벤트로 인해 타일맵이
						변경될 수도 있다.
 이벤트 트리거 노드	: 이벤트 트리거가 설정된 타일맵 하나하나를 나타내며 여러 개의 노드가 리스트를 이루고 있다. 타일맵이 변경되는 이벤트가 발생할
						경우 다음 순서의 타일맵으로 변경된다.

편집 방식
 편집 순서 : 편집 시작 -> 타일맵 사이즈(개수 X 개수) 결정 -> 편집

단축키 모음
 Ctrl + S	: 전체 세이브
 Ctrl + L	: 타일맵 로드
 Ctrl + C	: 타일맵 좌측 상단 위치 변경
 Ctrl + T	: 이벤트 트리거 모드로 변경
 Ctrl + F	: 이벤트 트리거 수정 모드로 변경
 Ctrl + N	: 이벤트 트리거 노드 세이브
 F1			: 타일 배치 모드로 설정
 F2			: 오브젝트 배치 모드로 설정
 F3			: 타일 옵션 설정 모드로 설정
 F4			: 이벤트 트리거 설정 모드(이벤트 트리거 모드에서만 사용 가능)로 설정
 1 ~		: 각 모드 별 선택지 고르기
