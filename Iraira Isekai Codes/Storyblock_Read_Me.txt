"열받는 이세계(2020년)" 프로젝트의 전에 진행한 개인 프로젝트에 사용된 스토리블록 시스템의 매뉴얼이 적힌 문서입니다.
이 스토리블록 시스템은 추후의 Unity 프로젝트들에서 쓰이는 스토리블록 시스템의 모체입니다.
"イライラ異世界(2020年)"プロトタイプの前に進行した個人プロトタイプに使用されたストリーブロックのシステムのマニュアルが書かれた文章です。
このストリーブロックシステムは後のUnityプロジェクトたちで使われるストリーブロックシステムの母体であります。

上から順番に

ストリーブロック使用のマニュアル

1. ストリーブロックのシステム
ゲイムシーン中での活用			: ストリーブロックベクター
ストリーブブロックの構成要素	: ストリーノードベクター
ストリーノードの構成要素		: サウンドキーベクター + カットシンパンネル(Panelオブジェクト) + キャラクターパンネル(Panelオブジェクト)ベクター
ストリーブロックの作動方式		: 条件の満足 -> ストリーブロックの活性化 -> Story_Block::Inputでストリーノードを渡すたびにStory_Block::Goto_Next_Nodeを呼び出す
									-> Story_Block::Goto_Next_Nodeがストリーノードベクターが終わった信号を返すと
									Gamenovel_Ingame_Scene::Goto_Next_Storyblockを呼び出す -> 非活性化されていた客体たち再び活性化。

2. ストリーブロックの生成時の注意点
複数のストリーブロックを生成するとき使用するストリーブロック1つ以外は全部Set_Enable(False)すること。(基本的にm_enableはTRUEだ)
STORYBLOCK_NEXTNODE_EFFECT_KEYに登録されたサウンドはストリーノードが渡るたびに自動的に再生するためユーザーが毎回ノードに追加する必要はない。

だと書いております

스토리블록 사용 매뉴얼

1. 스토리블록 시스템
게임 씬 내에서의 활용	: 스토리블록 벡터
스토리블록 구성 요소	: 스토리노드 벡터
스토리노드 구성 요소	: 사운드 키 벡터 + 센텐스 패널 + 컷씬 패널(Panal 오브젝트) + 캐릭터 패널(Panal 오브젝트) 벡터
스토리블록 작동 방식	: 조건 만족 -> 스토리블록 활성화 -> Story_Block::Input에서 스토리노드 넘길 때마다 Story_Block::Goto_Next_Node 호출
							-> Story_Block::Goto_Next_Node가 스토리노드 벡터가 끝났다는 신호를 반환하면
							Gamenovel_Ingame_Scene::Goto_Next_Storyblock 호출 -> 비활성화 되었던 객체들 다시 활성화.

2. 스토리블록 생성 시 주의할 점
여러 개의 스토리블록을 생성할 때 사용할 스토리블록 하나 외에는 전부 Set_Enable(FALSE) 할 것.(기본적으로 m_enable은 TRUE이다)
STORYBLOCK_NEXTNODE_EFFECT_KEY에 등록된 사운드는 스토리노드가 넘어갈 때마다 자동으로 재생되니 사용자가 매 노드마다 추가할 필요 없다.
